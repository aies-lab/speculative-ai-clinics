# Reuse guide

Default license
This repository is licensed under Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0) unless a file or folder states otherwise.

What CC BY-SA 4.0 allows
You may copy, share, and adapt the materials for any purpose, including commercial use, as long as you:
- Give appropriate attribution
- Indicate if you made changes
- License your adaptations under CC BY-SA 4.0

How to attribute
Include the following wherever practical:
- Title of the material
- Author/organization (Grounded Speculative AI Clinics · UW Information School)
- Source (this repository)
- License (CC BY-SA 4.0)
- Changes (if any)

Example attribution
"Speculative Accountability Packet Template" by Grounded Speculative AI Clinics · UW Information School, licensed under CC BY-SA 4.0. Changes: adapted formatting and removed partner-specific sections.

Partner-approved or restricted materials
Some materials may be shared only with partner approval or may be de-identified by default. If a file or folder includes a different license or access note, that notice overrides the default CC BY-SA 4.0 license.

Questions
If you are unsure about reuse, email the project team at belencsf@uw.edu.
